# 📊 GitTrending-Daily

**每日 GitHub Trending 热门仓库中文日报 | Daily Chinese Report of GitHub Trending Repositories**

自动抓取 GitHub Trending 热门项目，生成中文日报，通过 GitHub Actions 每天自动更新。每天早上看一遍，了解技术圈最新动态！

## ✨ 特色

- 📥 **每日自动抓取** GitHub 最热门的 20 个仓库
- 🇨🇳 **中文介绍**，阅读友好，快速浏览
- 📄 **GitHub Pages** 在线浏览，无需安装
- 🔄 **每天 UTC 0:00** 自动更新，雷打不动
- 🏷️ **标签信息**，快速了解项目分类
- 🆓 **开源免费**，Fork 即用

## 👀 在线查看

👉 **[qiang-sens.github.io/GitTrending-Daily/](https://qiang-sens.github.io/GitTrending-Daily/)**

无需安装，打开即看每日最新 GitHub 热门项目。

## 🚀 自己部署

Fork 这个仓库，GitHub Actions 会自动启用，每天抓取最新的 Trending 数据推送到你的 Pages。

```bash
# 1. Fork 本仓库
# 2. 进入 Settings > Pages，选择 GitHub Actions 部署
# 3. 等待第一次 Actions 运行完成
# 4. 访问 https://你的用户名.github.io/GitTrending-Daily/
```
