# 📊 GitTrending-Daily

**每日 GitHub Trending 热门仓库中文日报**

自动抓取 GitHub Trending 热门项目，生成中文日报，通过 GitHub Actions 每天自动更新。

## ✨ 特色

- 📥 自动抓取 GitHub 最热门仓库
- 🇨🇳 中文展示，阅读友好
- 📄 GitHub Pages 在线浏览
- 🔄 每天 UTC 0:00 自动更新
- 🆓 开源免费
